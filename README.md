# What is it?
OSCRaccoon is an app for Quest standalone that can display the current song you're listening to, as well as custom cycling messages and your local time to your VRChat chatbox through OSC without a PC! Download it [HERE](https://github.com/RechoRaccoon/OSC-Raccoon/releases/download/v1.0/OSCRaccoon.apk)!
